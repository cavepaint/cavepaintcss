Text padding

The .text-padding class is a smaller padding meant for text. It doesn't respond at all with .responsive.

It can be used to pad bits of text that need separation because of colors:

Example text

Example text

Notice the second paragraph, which has the text-padding class, has a border between the edge of the color and the text.

The project's default text padding also makes a great buffer for the edge of the screen on mobile.

Text margin

The .text-margin composable is the same size as the text padding, but functions as a margin.

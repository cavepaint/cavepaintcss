CSS, now in HTML flavor

Most of Cavepaint's composable classes are just CSS properties.

The pattern here is simple: The property name is also the class, and applying the class applies the project's default attribute for that property.

For example, adding the line-height class sets line-height to 1.8, the project's default. If you use the Less version, you can also change the project defaults.

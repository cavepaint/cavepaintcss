Details

  • class: red, green, blue, etc.
  • settings: color-strength, ambient-mix
  • defaults: none
composable many classes

Examples

Give elements color by adding a color class:

<button>
  A button!
</button>

<button class="red">
  A button!
</button>

Project colors are generated from one parameter, the @base-color.

base-color

Here, it's given to you. But in your own project you can define it to be whatever you want.

Light and dark

Color classes compose with the .light and .dark classes:

<button class="light green">
  A button!
</button>

<button class="dark green">
  A button!
</button>

Color helpers

Helper classes warm, neutral, cool, bright and dull compose with colors to change their feeling:

Because these use background-image, only one color helper from this section can be used at a time.

.texture

All colors compose with .texture too:

A red headline

<h3 class="red text-padding">
Wow!
</h3>

A textured red headline

<h3 class="red texture text-padding">
Wow!
</h3>

Patterns can be invoked in any color using the .pattern() mixin.

They aren't available out-of-the-box in the CSS version.

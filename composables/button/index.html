Examples

It's a button!

<button class="button">Click me!</button>

You can add the button class in the HTML or extend it in Less to include the elements you want to consider buttons (like "button" and "input[type='submit']").

Buttons compose with the other composable classes:

class="button dark green"

class="button padding texture skyblue"

class="button action border no-shadow"

class="button big border charcoal monospace"

class="button small yellow"

class="button inner-gutter texture big white"

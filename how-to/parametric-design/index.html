Design through parameters

Cavepaint aims to be as non-prescriptive as possible.

Part of that means not prescribing colors. The base color of a Cavepaint theme can be any color. In order to avoid prescribing a color, and ending up with every Cavepaint theme looking the same, this demo site includes CSS files with randoe base colors.

Here's the current base color:

@base-color

Other parameters

In addition to color, Cavepaint offers other parameters through the Less version in settings.less. Some of the basics:

  • .font-size
  • .line-height
  • .padding
  • .margin
  • .gutter

Switching out Cavepaint parameters

Rebranding or interface overhauls can sometimes call for massive site-wide changes. Cavepaint is designed with these types of changes in mind. The entire color scheme of any Cavepaint site can be changed by changing the base color.
